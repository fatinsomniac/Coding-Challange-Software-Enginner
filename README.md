# JUDUL PROYEK
Piramida Fibonacci

# Algoritma
### Algoritma dengan Struktur Bahasa Indonesia :
```text
1.  Mulai
2.  Buat fungsi print_deret_fibonacci dengan parameter variable jumlah
3.  Buat variable angka_sebelumnya dan angka sekarang dengan nilai 0 dan 1
4.  Buat variable hasil dan kasih penugasan untuk angka_sekarang dengan type string
5.  Lakukan pengulangan dari 0 sampai sampai var jumlah - 1  kurang = 0
6.  Buat var output unruk menyimpan hasil dari (var angka_sekarang + var angka_sebelumnya)
7.  Buat var hasil untuk menyimpan var hasil dan var output
8.  Buat var angka_sebelumnya untuk menyimpan var angka_sekarang saat ini
9.  Buat var angka_sekarang untuk menyimpan hasil var output
10. Pengulangan berhenti ketika nilai 0 lebih (var jumlah -1)
11. Cetak hasil
12. Buat fungsi piramida_fibonacci dengan parameter variable tingkat
13. Lakukan pengulangan dari 1 sampai var tingkat + 1 kurang 1
14. Tampilkan fungsi print_deret_fibonacci dengan parameter var i
15. Pengulangan berhenti apabila nilai var i lebih besar dari pada (var tingkat+1)
16. Tentukan nilai untuk menentukan jumlah baris pada piramida fibonacci
17. Selesai
```

### Algoritma dengan Pseudocode :
```text
Algoritma membuat Piramida Fibonacci

Deklarasi :

var angka_sebelumnya => 0
var angka_sekarang => 1

Deskripsi :

fungsi print_deret_fibonacci dengan parameter var jumlah
hasil => angka_sekarang
for i=>0, i < jumlah-1, i++

output => angka_sekarang + angka_sebelumnya
hasil => hasil 'output'

angka_sebelumnya => angka_sekarang
angka_sekarang => output

cetak kembali hasil

fungsi piramida_fibonacci dengan parameter var tingkat
for i=>1, i < tingkat+1, i++
tampilkan fungsi pertama(i)

tentukan baris piramida fibonacci(8)
```
